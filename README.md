# Como jogar

• Baixe o .exe do jogo nesse link https://github.com/VictorG-028/Projeto-de-LPI-Moon-Buggy/raw/master/Projeto%20de%20LPI%20-%20Moon%20Buggy/bin/Release/Projeto%20de%20LPI%20-%20Moon%20Buggy.exe
